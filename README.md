# Heat Is a Debt the Ocean Has Been Paying

Static website for a Bow Seat Ocean Awareness Contest Interactive & Multimedia entry.

Open `index.html` in a browser, or host this folder as a static site. The experience uses only HTML, CSS, and JavaScript. It does not load external images, fonts, audio, libraries, or external media assets.

## What to review before any contest submission

Before submitting, review the final version carefully:

- Review visible text in your own English.
- Confirm the visual system, colors, timing, and interaction match the intended story.
- Take at least one progress screenshot while editing the code or visuals.
- Write the concept description and reflection using the templates in this folder.
- Keep a works-cited list for facts, inspiration, and any media you add.

## Files

- `index.html`: 9-chapter scrollytelling structure and source links.
- `styles.css`: responsive UI, fixed controls, chapter text, and final interaction styling.
- `script.js`: code-native canvas world engine, scroll transitions, metrics, pointer effects, and reciprocity interaction.
- `final-submission-plan.md`: final submission writing and screenshot plan.
- `submission-package/`: curated application field guide and five recommended screenshot files.
- `concept-description-template.md`: fill-in guide for the short description of use.
- `reflection-template.md`: fill-in guide for the required 100-300 word reflection.
- `works-cited.md`: source list starter.
- `authorship-checklist.md`: validity checklist before submitting.
- `screenshots/`: place final screenshots here.

## Suggested submission screenshots

Bow Seat allows up to 5 images or supporting files for Interactive & Multimedia. Capture:

1. Opening inland heat scene.
2. Breath layer with plankton/oxygen particles.
3. Coral city with animals.
4. Bleaching or no-water future.
5. Reciprocity restored state or a progress screenshot from your editor while you personally edit the work.

## Local testing

If opening the file directly is enough:

```text
Open index.html in your browser.
```

If you want a local URL:

```powershell
python -m http.server 5173
```

Then open `http://127.0.0.1:5173`.

## Curated Submission Package

Use `submission-package/application-fields.md` and `submission-package/today-submit-checklist.md` when filling the form. The curated screenshots are separated from older QA screenshots so you do not accidentally submit an outdated image.
